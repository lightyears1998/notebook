# Gulp

- [Docs](https://gulpjs.com/docs/en/getting-started/javascript-and-gulpfiles)

## 安装

```sh
npm install --global gulp-cli
npm install --save-dev gulp
```

gulpfile.js

```sh
npx -p touch nodetouch gulpfile.js
gulp --help
```

gulpfile.babel.js

```sh
npm install --save-dev @babel/register
```
