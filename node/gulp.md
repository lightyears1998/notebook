# Gulp

- [Docs](https://gulpjs.com/docs/en/getting-started/javascript-and-gulpfiles)

## 安装

```sh
yarn global add gulp-cli
yarn add --dev gulp
```

gulpfile.js

```sh
npx -p touch nodetouch gulpfile.js
gulp --help
```

gulpfile.babel.js

```sh
yarn add --dev @babel/register
```
