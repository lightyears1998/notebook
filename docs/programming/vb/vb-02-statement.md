# Visual Basic 语句

## 顺序语句

语句的末尾不需要分号。

### 严格模式

```vb
Option Explicit
```

### 声明语句和赋值语句

```vb
Dim Count As Integer
Count = 1
```

## 分支语句

```vb
If booleanExpression Then
    ' ...
ElseIf booleanExpression Then
    ' ...
Else
    ' ...
End If
```

## 循环语句

```vb
While booleanExpression
    ' ...
Wend
```
