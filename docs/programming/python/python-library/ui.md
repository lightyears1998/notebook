## QT Python Binding

- `pyside2` LGPL
- `pyqt` GPL

通过这些链接了解基础知识：

- <https://realpython.com/python-pyqt-gui-calculator/> More than a "calculator"

入门之后，应该参考 QT 的[笔记](../../../platform/qt/README.md)和官方文档。
