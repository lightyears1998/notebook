# PyTorch

基本操作思路：

1. 定义继承于 nn.Module 的神经网络模型，并定义模型的 forward 方法。
2. 定义继承于 nn.Module 的误差函数，并定义误差函数的 forward 方法。
3. 选择 optimizer。
4. 训练。
