# 方法

为了珍惜使用时间，在可感知的各种时间尺度上开展三阶段行动模式。

- **三阶段模式** 准备 / 执行 / 整理
- **时间尺度** 人生 / 短期 / 自然日

## 人生的三阶段模式

- **准备** 形成理想。
- **执行** 通过短期的三阶段模式落实。
- **整理** 根据自我和环境变化及时调整理想。

## 短期时间的三阶段模式

- **准备** 形成短期行动计划。
- **执行** 在各个自然日中落实。
- **整理**

### 短期行动计划

1. 指定辅助统计的工具。
2. 指明需要以数据形式跟踪的指标。

## 自然日的三阶段模式

- **准备** 形成 *自然日一般日程规划* 与 *待办计划*。
- **执行**
- **整理**

### 自然日一般日程规划

警戒时段 睡前1.5小时
清空时段 睡前0.5分钟

### 自然日待办计划

在一般日程规划的基础上填入具体行动计划。
