## 习惯

- **盘古之白** 在等宽渲染的上下文中，全角字符（不包括全角中文标点符号）与半角字符之间使用一个半角空格分开。
- **延迟回复** 在需要回执的通信中，如果要延后特定消息的处理，须给予消息发送方延迟处理的响应，并将消息挂入任务队列。
